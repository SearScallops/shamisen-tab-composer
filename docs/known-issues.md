# Known Issues

This document tracks known problems in Shamisen Tab Composer Alpha 0.1.

## Current Known Issues

- The app is currently focused on Windows desktop testing.
- Sheet layout may need improvement for long songs.
- PDF export uses the current visual sheet capture, so very wide sheets may be difficult to print neatly.
- Mobile layouts are not ready yet.
- Technique notation may change as more shamisen notation rules are added.
- File format may change before the stable release.

## Not Bugs

These are expected behaviors:

- Lyrics can only be added under an existing note or rest.
- Suri slides must connect two notes on the same string.
- Repeat marks cannot be placed over measures that already contain notes or rests.
- One-measure repeats cannot be placed in measure 1.
- Two-measure repeats cannot be placed before measure 3.
