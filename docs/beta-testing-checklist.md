# Beta Testing Checklist

Use this checklist before sharing Shamisen Tab Composer with testers.

## Basic Song Setup

- [ ] Change the song title
- [ ] Change tuning between Honchoshi, Niagari, and Sansagari
- [ ] Change BPM
- [ ] Change measure count
- [ ] Change zoom level

## Note Input

- [ ] Place notes on String 1
- [ ] Place notes on String 2
- [ ] Place notes on String 3
- [ ] Try Whole, Half, Quarter, Eighth, and Sixteenth rhythms
- [ ] Try different tab numbers
- [ ] Select an existing note and edit its tab number
- [ ] Select an existing note and edit its rhythm
- [ ] Select an existing note and edit its technique
- [ ] Confirm overlapping notes are blocked

## Techniques

- [ ] Test left-hand techniques
- [ ] Test right-hand techniques
- [ ] Confirm Oshibachi shows 押 and ⌈ above the note
- [ ] Confirm Suberi shows 滑 and ⌈ above the note

## Rests

- [ ] Place rests on each string
- [ ] Try different rest rhythms
- [ ] Confirm rests cannot overlap notes
- [ ] Confirm notes cannot overlap rests

## Suri Slides

- [ ] Add a Suri slide between two notes on the same string
- [ ] Remove a Suri slide
- [ ] Confirm Suri cannot connect notes on different strings
- [ ] Confirm deleting a note also deletes attached Suri slides

## Lyrics

- [ ] Add lyrics under a note
- [ ] Add lyrics under a rest
- [ ] Edit an existing lyric
- [ ] Delete a lyric
- [ ] Confirm lyrics cannot be placed without a note or rest

## Section Labels

- [ ] Add section labels like Intro, Verse, and Chorus
- [ ] Edit a section label
- [ ] Delete a section label
- [ ] Confirm labels stay aligned with measures

## Simile Repeats

- [ ] Add a one-measure repeat
- [ ] Add a two-measure repeat
- [ ] Remove repeat marks
- [ ] Confirm repeat marks cannot overlap notes or rests
- [ ] Confirm one-measure repeat cannot be placed in measure 1
- [ ] Confirm two-measure repeat cannot be placed before measure 3

## Undo and Redo

- [ ] Undo note placement
- [ ] Undo rest placement
- [ ] Undo Suri slide changes
- [ ] Undo lyric changes
- [ ] Undo section label changes
- [ ] Undo repeat mark changes
- [ ] Redo actions after undoing

## Save and Load

- [ ] Save a song
- [ ] Load the saved song
- [ ] Confirm all song data loads correctly
- [ ] Delete a saved song from the Load window
- [ ] Open the song library folder

## Import and Export

- [ ] Export JSON
- [ ] Import JSON
- [ ] Export PNG
- [ ] Export PDF
- [ ] Open the export folder
- [ ] Reveal the last exported file

## Keyboard Shortcuts

- [ ] Ctrl + S saves
- [ ] Ctrl + O loads
- [ ] Ctrl + N starts a new song
- [ ] Ctrl + Z undoes
- [ ] Ctrl + Y redoes
- [ ] Ctrl + P exports PDF
- [ ] Ctrl + Shift + P exports PNG
- [ ] Delete removes selected note
- [ ] Escape clears selection
- [ ] Ctrl + 1 through Ctrl + 7 switch tools

## Final Beta Check

- [ ] App does not crash during normal use
- [ ] Status messages are clear
- [ ] Exported files open correctly
- [ ] Saved files are easy to find
- [ ] Help screen is understandable for a beginner
- [ ] Tester understands this is an Alpha version
